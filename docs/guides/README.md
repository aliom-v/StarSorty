# Guides 导航

`guides/` 目录存放稳定的说明文档，适合日常查阅和协作交接。

### 部署与维护

## 文档列表

- `project-structure.md`：项目目录、关键入口文件、运行关系
- `user-manual.md`：日常使用路径、管理操作与常见问题
- `api-reference.md`：API 接口、鉴权、任务模型、调用示例
- `configuration.md`：环境变量、配置优先级、生产配置建议
- `deployment-operations.md`：部署流程、备份恢复、常见运维动作

## 使用约定

- 偏“长期有效”的说明写在 `guides/`。
- 偏“阶段性计划”的内容写在 `../roadmap/`。
- 如果某篇文档内容显著增多，优先拆新文件而不是继续堆在 `README.md`。
