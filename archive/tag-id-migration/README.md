# Tag ID Migration Archive

该目录保存一次性 `tag_id` 迁移相关资产，作为历史记录保留，不参与当前开发、测试、部署与 CI 流程。

## 包含内容

- `migrate_tag_ids.py`：历史迁移脚本
- `rules.v2.json`：当时使用的迁移规则快照
- `tag_id_migration_report.json`：迁移执行报告样例

## 说明

- 当前仓库运行路径、文档、CI 与常用脚本均不再依赖这些文件。
- 若未来需要追溯迁移过程或复盘历史数据，可在此目录查看。
- 如需再次做类似迁移，建议基于当前 taxonomy 与数据模型重新生成脚本，而不是直接复用这里的历史资产。
